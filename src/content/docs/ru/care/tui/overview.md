`care` (без подкоманды) запускает Textual TUI. Он открывается прямо в **чате** —
транскрипте в стиле Claude Code с полем ввода внизу и переключателем режима над ним.

## Интерфейс чата

- **Введите задачу** на естественном языке → MAGE генерирует цепочку, CARL запускает её, ответ
  выводится инлайн.
- **Введите [слэш-команду](/ru/care/slash-commands/overview/)** (`/help`, `/library`, …)
  для операций вне чата.
- **Переключатель режима** над полем ввода переключает [Interactive ↔ Production](/ru/care/workflows/modes/) —
  по умолчанию выбран Interactive.
- **Загляните в цепочку** за любым ответом: из чата доступны [средства работы с DAG цепочки](/ru/care/tui/dag/),
  чтобы пройтись по графу, а **плашка Artifacts** собирает файлы, которые создал запуск.
- Новым пользователям предлагается набрать `/tour` — пятишаговое руководство.

### Как разворачивается запуск

Пока MAGE генерирует и CARL исполняет, транскрипт показывает **трассировку этапов** —
строки `▶ Понятная метка…`, которые завершаются `✓`, с `⎿`-подстроками для деталей.
Строка статуса отслеживает использование токенов в реальном времени.

### Прикрепление файлов

Ссылайтесь на файлы инлайн с помощью `@<путь>` — например, `summarise @report.pdf`. PDF
извлекаются как текст; изображения встраиваются для моделей с поддержкой зрения. Пути с пробелами
можно взять в кавычки: `@"My Notes.md"`.

## Глобальные клавиши

| Клавиша | Действие |
| --- | --- |
| `Ctrl+P` | Палитра команд (fuzzy по командам + сохранённым сущностям). |
| `Ctrl+B` | Панель задач (работающие воркеры). |
| `Ctrl+K` | Каталог возможностей (навыки / MCP / инструменты). |
| `?` | Справка — туториал + все привязки для активного экрана. |
| `Ctrl+C` | Выход. |

Каждый экран добавляет свои привязки — нажмите `?` для набора, отфильтрованного по текущему экрану.
