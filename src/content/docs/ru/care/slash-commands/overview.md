В чате всё, что **не является** `/командой`, отправляется в MAGE как
описание задачи. Слэш-команды — это операции вне чата: открыть экраны, управлять
сессией, переключать режимы.

:::tip
Введите `/help` в TUI для живого списка, отфильтрованного по вашему текущему режиму, и `/tour`
для пятишагового руководства.
:::

## Основные

| Команда | Что делает |
| --- | --- |
| `/help` | Показать справку по командам. |
| `/tour` | Пятишаговое интерактивное руководство. |
| `/mode [ad-hoc\|production]` | Переключить или показать текущий [режим чата](/ru/care/getting-started/overview/). |
| `/settings` | Открыть экран Настроек. |
| `/new` | Начать новый чат (сбросить историю Ad-Hoc). |
| `/clear` | Очистить транскрипт. |
| `/quit` | Выйти из MAESTRO CARE. |

## Цепочки и библиотека

| Команда | Что делает |
| --- | --- |
| `/artifacts` | Просмотр цепочек, сгенерированных в этом чате. |
| `/library` | Открыть библиотеку сохранённых агентов. |
| `/run <chain_id>` | Открыть сохранённую цепочку для выполнения. |
| `/revise [<id>] <change>` | Редактировать цепочку на естественном языке (режим редактирования MAGE → сохраняет новую версию). |
| `/marketplace "<query>"` | Поиск общих листингов `agent_skill`. |
| `/memory ...` | Просмотр хранилища Memory. |
| `/evolution [<run_id>]` | Открыть дашборд эволюции / запуск. |

## Сессия и транскрипт

| Команда | Что делает |
| --- | --- |
| `/resume [latest\|<file>]` | Восстановить сохранённую сессию. |
| `/sessions` | Список / управление сохранёнными сессиями. |
| `/history [N]` | Список последних промптов + сохранённых цепочек. |
| `/edit [N\|list]` | Перередактировать прошлый промпт пользователя (нумерация с 1). |
| `/multi` | Открыть многострочный компоновщик задач. |
| `/branch [name\|list\|switch <id>\|delete <id>]` | Контрольные точки транскрипта. |
| `/blocks [copy\|save N [path]]` | Список или действие над блоками кода. |
| `/export <md\|mdx\|json\|html> [path]` | Сохранить транскрипт на диск. |
| `/remember <text>` | Сохранить заметку в долгосрочный контекст. |

## Диагностика и UI

| Команда | Что делает |
| --- | --- |
| `/status` | Сводка состояния сессии. |
| `/cost` | Открыть дашборд стоимости. |
| `/runs` | Открыть экран запусков. |
| `/logs` / `/log [level] [module]` | Открыть / хвост лога приложения. |
| `/profile` | Открыть экран профиля. |
| `/sandbox` | Настройки доверия к песочнице. |
| `/subagents [clear]` | Отрисовать захваченные события шагов CARL в виде дерева. |
| `/theme [name]` | Список или переключение темы UI. |
| `/imgpreview [status\|<path>]` | Поддержка терминальной графики / последовательность изображений. |
| `/voice [status\|transcribe <path>]` | Транскрипция Whisper. |

См. [команды Production](/ru/care/slash-commands/production/) для команд
датасет / эволюция / публикация, доступных в режиме Production.
